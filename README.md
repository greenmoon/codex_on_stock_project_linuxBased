# codex_on_stock_project_linuxBased

Stock monitoring GUI project based on `jb_A411B_stock_AI_v36b.py`.

The app is a PyQt5 desktop tool for browsing selected Taiwan stock symbols and viewing market-derived indicators.

## Main File

- `jb_A411B_stock_AI_v36b.py`

## Requirements

Install dependencies with:

```bash
python3 -m pip install -r requirements.txt
```

## Run

```bash
python3 jb_A411B_stock_AI_v36b.py
```

The app fetches market data from TWSE and TPEX network endpoints, so an internet connection is required.
